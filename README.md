# Appii Xibb-un-Nabawi (Qoricha Nabiyyii) 🌿

Appiin kun gorsa fayyaa fi qoricha aadaa kan barsiisa beekamaa **Ibnul Qayyim Al-Jawziyya** irratti hundaa'e bifa salphaan Afaan Oromootiin dhiyeessa.

## 🌟 Mul'ata Pirojeektii
Beekumsa fayyaa kan barsiisa amantii fi saayinsii aadaa walitti qabate, karaa ammayyaa fi teeknoolojii fayyadameen uummata keenya biraan gahuu.

## 📖 Qabiyyee Appichaa (Features)
* **Fayyaa Qaamaa:** Gorsa hirribaa, uffataa, dhugaatii fi nyaataa.
* **Yaala Bowwoo Mataa:** Sababoota bowwoo mataa 20 fi qorichoota isaanii.
* **Dandeettii Sammuu:** Akkaataa irraanfannaa itti ittisan fi qalbi-qabeessummaa itti dabalan.
* **Gorsa Jaalalaa:** Ishqii fi dhiphuu sammuu irraa itti fayyaman.
* **Offline Access:** Odeeffannoo hunda interneetii malee dubbisuun ni danda'ama.

## 🚀 Teeknoolojii Fayyadamne
* **Framework:** Flutter (Android & iOS)
* **Language:** Dart
* **Storage:** JSON / SQLite (Odeeffannoo itti kuusuuf)
